# Team Helper Telegram Bot

Простой Telegram-бот на Python для внутренней команды.

## Возможности

- Отвечает на базовые вопросы о компании, проекте и команде
- Хранит личные дела (контакты) коллег
- Удаляет контакты по номеру
- Ищет сотрудников по имени или отделу в `employees.csv`
- Хранит данные в `contacts.json` отдельно для каждого пользователя (`user_id`)
- Содержит базовую обработку ошибок

## Команды

- `/start` — приветствие и список возможностей
- `/add_contact` — добавить личное дело коллеги
- `/add` — алиас для добавления (дополнительно)
- `/list_contacts` — показать список личных дел
- `/delete` — удалить личное дело по номеру
- `/employees` — найти сотрудников по имени или отделу из CSV
- `/cancel` — отменить текущий диалог

## CSV сотрудников

Бот читает файл `employees.csv` в корне проекта.

Ожидаемая структура колонок:

```csv
name,department,role,email
```

Пример:

```csv
name,department,role,email
Иван Петров,Engineering,Backend Developer,ivan.petrov@company.com
Мария Смирнова,HR,HR Manager,maria.smirnova@company.com
```

Как работает `/employees`:

1. Отправьте команду `/employees`.
2. Введите имя сотрудника **или** отдел.
3. Бот покажет все совпадения (по части текста, без учета регистра).

## Как запустить

1. Установите Python 3.10+.
2. Откройте терминал в папке проекта:

```bash
cd d:\Projects\team_helper_bot
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```

4. Создайте `.env` на основе примера:

```bash
copy .env.example .env
```

5. В файле `.env` укажите токен Telegram-бота:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
```

6. Запустите бота:

```bash
python bot.py
```

## Примечания

- Файл `contacts.json` создается автоматически при первом сохранении.
- Если `employees.csv` не найден или содержит неверные колонки, бот покажет понятную ошибку.
- Если JSON поврежден, бот не упадет: будет использован пустой список.
- Для реальных ежедневных дайджестов можно добавить планировщик (например, APScheduler).
